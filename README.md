This repository contains the source code for the IE4476 Image Processing & Computer Vision project.
The project explores dimensionality reduction methods such as PCA, LDA, and various classifiers including k-NN, Logistic Regression, and Mahalanobis Distance using the Fashion-MNIST dataset.
